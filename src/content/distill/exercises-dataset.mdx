如果你正在做健身 ML 或開發運動 App，這個開源資料集值得關注。hasaneyldrm/exercises-dataset 蒐集了上千個健身動作，每個都有縮圖和動畫 GIF，最重要的是提供 9 種語言逐步教學。這種多語言規格在健身資料集裡很少見。

結構化是它的強項——每筆記錄除了目標肌群，還細分到器材類型、協助肌群和次要肌群，這個層級的標註非常適合訓練推薦系統或動作辨識模型。JSON Schema 可驗證資料格式，不用自己寫程式檢查。開發體驗也做得很完整——index.html 是互動式動作瀏覽器，打開就能篩選；setup.html 可一鍵產出 SQL 或 API code。

LogPress AI 訓練追蹤 App 已採用。對資料工程師來說，這種品質的 labeled dataset 欄位完整、格式統一，拿來練手或當 baseline 都合適。
