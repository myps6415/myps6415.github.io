WrenAI 是一個開源的生成式 BI 引擎，它讓 AI Agent 不只會寫 SQL，還能直接幫你生成可分享的儀表板並部署上線。支援超過 22 種資料源。

傳統 BI 工具是人在點擊操作，LLM Agent 是胡亂猜測 SQL。WrenAI 的厲害之處是給 Agent 一個「情境層」：包含業務語意、公司定義的規則、成功的查詢範例，還有公司內部的文件知識。Agent 不只看得到資料表結構，還知道什麼叫「營業額」。

運作方式很簡單：裝好 CLI、下載 stub 檔案，就能用自然語言叫 Agent 設定資料庫、寫出受治理的 SQL，最後把結果變成互動式儀表板部署到 Vercel。

我覺得最實用的是「治理」機制。WrenAI 把 SQL 生成過程變成可審視、可追蹤的，錯誤會附上修正提示。這種「對的東西才給過」的機制，才是讓 AI 真正能上工的原因。

如果你的團隊已經在用 Claude Code、Cursor，需要可信的資料分析結論，WrenAI 是目前最完整的開源解方。
